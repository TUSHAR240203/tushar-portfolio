# 🚀 Tushar Sharma | Developer Portfolio

A modern and responsive developer portfolio built with **React, Vite, TypeScript and Tailwind CSS** showcasing my projects, skills and experience. The portfolio includes smooth animations, a contact form integration and a downloadable resume.

---

## 🌐 Live Website

[https://tushar-portfolio.vercel.app](https://tushar-portfolio-amber.vercel.app/#)

---

## 🛠 Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- EmailJS

---

## ✨ Features

- Modern responsive UI
- Dark / Light theme toggle
- Smooth animations using Framer Motion
- Project showcase section
- Contact form with EmailJS integration
- Resume download option
- Mobile friendly navigation
- Clean and minimal design

---

## 📂 Installation

Clone the repository

```bash
git clone https://github.com/TUSHAR240203/tushar-portfolio.git
```

Navigate to project folder

```bash
cd tushar-portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

## ⚙️ Build for Production

```bash
npm run build
```
